Text Generation

Using LSTM

We have done a Text Generation project using LSTM which is an artificial recurrent neural network architecture used in the field of deep learning. We worked with the dataset of US President's speech. We gave a sequence of characters from this data to train a model to predict the next characters. Our text generation model is based on simple Recurrent Neural Network (RNN) based language model trained on the text of the target author. The RNN learns to predict the next word given the previous context.We aim to use current state of the art recurrent neural network techniques to generate text in that same style.
